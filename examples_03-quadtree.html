Quadtree parametrisation¶

The Quadtree subsamples the InSAR displacement in order to have a reduced and thus more manageable dataset for displacement modelling. The tree consists of QuadNode s, a state or slice of the tree is denoted as leafs.

Four parameters characterize the quadtree and when a QuadNode is split:

  • epsilon/std threshold (epsilon)
  • Fraction of NaN values within (nan_allowed)
  • Maximum and minium dimension of the tile (tile_size_max and tile_size_min)

Note

All nodes of the Quadtree are built upon initialisation an instance.

The parametrization of the tree can be done through the Spool (GUI) or the python interface of Quadtree.

from sc import Scene
sc = Scene.import_data('test/data/20110214_20110401_ml4_sm.unw.geo_ig_dsc_ionnocorr.mat')

sc.quadtree  # this initializes and constructs the tree

sc.epsilon = .024
sc.nan_allowed = .9
sc.tile_size_max = 12000
sc.tile_size_min = 250

print(sc.reduction_rms)  # In units of [m]
>>> 0.234123152

for l in sc.quadtree.leafs:
    print l

# One may export the tree in a CSV format
sc.quadtree.export('/tmp/tree.csv')
